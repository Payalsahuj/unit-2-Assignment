b tag

The b tag in HTML is used to specify the bold text. The text which is written inside the b tag are displayed in bold size. This tag requires a starting as well as ending tag.

i tag

The i tag in HTML is used to display the text in italic style. This tag generally used to display technical term, phrase, the important word in different languages. This tag requires a starting as well as ending tag.

u tag

The u tag in HTML is used to underline the text. This tag requires a starting as well as ending tag.

select tag

The select tag in HTML is used to create a drop-down list. It contain option tag to display the available potion of drop down.

button tag

The button tag in HTML is used to create a button which is clickable in nature.Button can perform any function that is given to it by clicking on it.

hr tag

The hr tag in HTML is used as horizontal rule and is used to insert a horizontal rule or a thematic break in an HTML page to divide or separate document section.

br tag

The br tag insert a single line break. It is useful for writing addresses or poems. This tag is an empty tag which means that it has no end tag.
